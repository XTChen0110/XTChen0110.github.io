My First Selenium WebDriver Code

2020-09-20

What is Selenium WebDriver?

Selenium Webdriver is an open-source collection of APIs which is used for testing web applications(such as Chrome, Amazon, etc). The Selenium Webdriver tool is used for automating web application testing to verify that it works as expected or not.

What types of browsers the Selenium supports?

As we can tell, the Selenium Webdriver is powerful, so it mainly supports browsers like Firefox, Chrome, Safari and Internet Explorer. It also empowers you to execute cross-browser testing.

WebDriver also enables you to use a programming language in creating your test scripts (not possible in Selenium IDE).
